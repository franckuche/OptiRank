    À propos d'OptiRank

    OptiRank est une extension Chrome qui analyse les liens de votre page web pour détecter les liens brisés, les redirections et d'autres informations importantes.

    Comment utiliser OptiRank

    1. Analyser une page : Cliquez sur le bouton "Analyser les liens" dans le tableau de bord.
    2. Voir les résultats détaillés : Accédez à l'onglet "Résultats" pour voir tous les liens analysés.
    3. Configurer l'extension : Utilisez l'onglet "Paramètres" pour personnaliser le comportement de l'extension.
